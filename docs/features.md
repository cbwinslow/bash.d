This document lists the major features, planned improvements, and helpful notes for contributors.

## Core Features
- Unified CLI (`bashd`) for development and platform management
- Plugin system for data sources and integrations
- Data ingestion pipelines for public datasets (Congress.gov, GovInfo, ACS)
- Blog and content engine for publishing Markdown
- AI agents for automation, code generation, and data analysis
- Integrated testing and CI workflow
- Secure credential management via Bitwarden and MCP integration

## Planned / Near-term Features
- Expand connectors for more government and civic data sources
- Implement an API gateway and full-text search with indexing
- Add role-based access control and user management for the platform
- Build a polished data portal and post-processing pipelines
- CI/CD automation for releases and deployments

## Long-term Vision
- Reach enterprise-grade reliability and compliance (SOC 2 readiness)
- Provide advanced AI-driven analytics and personalization
- Create an ecosystem of community-maintained plugins and integrations

## Notes for Contributors
- Keep features scoped and small for early progress
- Update `docs/roadmap.md` and `docs/tasks.md` with any proposed items
- Document any new API endpoints in `docs/api.md`
