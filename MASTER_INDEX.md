# bash.d Master Index

Single jump-off point for the bash.d repository. This file groups the major
areas, generated indexes, and the fastest ways to search everything.

## Core Entry Points

- `README.md` - canonical overview and install instructions
- `QUICKSTART.md` / `QUICKSTART_NEW.md` - setup and usage shortcuts
- `bash_functions.d/README.md` - bash function library layout and tooling
- `docs/` - long-form docs and references

## Primary Directories (What Lives Where)

- `aliases/` - alias modules
- `completions/` - completion modules
- `bash_functions.d/` - function library (organized by category)
- `bash_aliases.d/` - user custom aliases
- `bash_env.d/` - environment snippets
- `bash_prompt.d/` - prompt configs
- `agents/` + `configs/agents/` - AI agent runtime + configs
- `docs/` - documentation hub
- `scripts/` - generators, setup, and utilities
- `tests/` - repo tests and validation
- `tldr/` - vendored tldr pages and tooling
- `tools/` - tool registry + helpers

## Search & Indexes (Generated + Primary)

- `bash_functions.d/script_inventory.json` - script inventory index
- `scripts/output/functions.json` - function index extracted from `bash_functions.d`
- `bash_functions.d/docs/man/` - man-style docs from script headers
- `bash_functions.d/docs/tldr/` - short TLDR summaries for bash functions
- `docs/tools_security_index.md` - security tools index
- `tldr/index.json` - tldr pages index (all platforms/languages)
- `$BASHD_STATE_DIR/module-index.json` - module index (generated by `lib/indexer.sh`)

## Fast Search Commands

- `rg "<term>"` - fastest full-text search
- `bashd_index_update` + `bashd_index_search "<term>"` - module index search
- `bash_functions.d/tools/bf_docs.sh lookup "<term>"` - header/doc search
- `bash_functions.d/tools/bf_docs.sh tldr <script>` - quick summary for a script
- `tldr <command>` - tldr pages for external commands
- `python -m tools.registry search "<term>"` - tool registry search

## Regenerate Index Artifacts

```bash
# Script inventory index
bash bash_functions.d/tools/script_inventory.sh \
  bash_functions.d \
  bash_functions.d/script_inventory.json

# Function index
python3 scripts/function_extractor.py \
  --input-dir bash_functions.d \
  --output-dir scripts/output \
  --format json

# TLDR summaries for custom functions
bash bash_functions.d/tools/tldr_generator.sh \
  bash_functions.d \
  bash_functions.d/docs/tldr

# TLDR master index
node tldr/scripts/build-index.js > tldr/index.json
```
