# Roadmap — bash.d

## Near-Term (0–3 months)
- Harden CI/CD and test coverage
- Add more government data connectors (Congress, GovInfo)
- Implement basic API gateway and search indexing
- Improve documentation and onboarding experience

## Mid-Term (3–12 months)
- Add advanced AI agents for content generation and analysis
- Achieve SOC2 readiness and better monitoring
- Develop a community plugin ecosystem

## Long-Term (12+ months)
- Production-grade multi-region deployment
- Advanced analytics and personalization features
- Growth of community contributions and external integrations

## How to contribute
- See `tasks.md` and `master_todo.md` for active work
- Propose a roadmap item in a PR and link to a design doc (SRS/arch)
