Convenience helpers for installing and running Qwen Code from the official repo.
Repo: https://github.com/QwenLM/qwen-code

Provided commands:
- qwen_code_install_latest: installs/updates the latest Qwen Code via pipx from GitHub.
- qwen_code_update: forces a reinstall/update from GitHub.
- qwen_code: wrapper that runs the available Qwen Code CLI (qwen-code/qwen/qwencode).
